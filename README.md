# נשמה וגוף - אתר ריפוי דרך תנועה

אתר לניהול אירועים ותוכן לעסק ריפוי דרך תנועה.

## התקנה

```bash
npm install
```

## הרצה

```bash
npm run dev
```

פתח [http://localhost:3000](http://localhost:3000) בדפדפן.

## ניהול תוכן

גש ל-[http://localhost:3000/admin](http://localhost:3000/admin) לניהול אירועים ותמונות.

### תכונות:

- **דף בית** - תצוגה יפה של האירועים והתוכן
- **פאנל ניהול** - הוספה, עריכה ומחיקה של אירועים ותמונות
- **העלאת תמונות** - העלה תמונות בקלות לאירועים או לשימוש כללי
- **עיצוב חם ובהיר** - עיצוב מותאם לעסק ריפוי

## מבנה הפרויקט

- `app/` - דפים ו-API routes
- `components/` - רכיבי React
- `lib/` - פונקציות עזר וניהול נתונים
- `public/uploads/` - תמונות מועלות
- `data/` - קבצי JSON לנתונים

## שפות וטכנולוגיות

- Next.js 14
- TypeScript
- Tailwind CSS
- React Hook Form
- Zod validation
